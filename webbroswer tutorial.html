How to create an ERC-20/BEP-20 Token without coding (web browser)

Install MetaMask

First of all you have to install the MetaMask-Extension in your browser. You can do this here. Then follow the instructions of MetaMask to either create an account or log into an already existing account.

Connect to the right Network

To connect to a network you have to click on the network at the top of the MetaMask-Extension and then choose the network you would like to connect to.
The following networks are Ethereum networks which can be used to create an ERC-20 token:

  • Ethereum Main Network
  • Ropsten Test Network
  • Kovan Test Network
  • Rinkeby Test Network
  • Goerli Test Network

Add a custom network

To create a BEP-20 token you have to connect to the Binance Smart Chain Mainnet or to the Binance Smart Chain Testnet.
To do this you have click at the network at the top and then choose "Custom RPC"

Type in the follwing to connect to the Binance Smart Chain Mainnet:
Network Name: Smart Chain
New RPC URL: https://bsc-dataseed.binance.org/
ChainID: 56
Symbol: BNB
Block Explorer URL: https://bscscan.com

Type in the follwing to connect to the Binance Smart Chain Testnet:
Network Name: Smart Chain - Testnet
New RPC URL: https://data-seed-prebsc-1-s1.binance.org:8545/
ChainID: 97
Symbol: BNB
Block Explorer URL: https://testnet.bscscan.com

Get BNB/ETH on your account

You need BNB or ETH on your MetaMask account to create a BEP-20 or ERC-20 token. If you want to deploy your token to the mainnet you have to send some BNB or ETH to your MetaMask address. You can copy this address right from the MetaMask-Extension.

If you want to deploy your token to a testnet you have to send some BNB or ETH to your MetaMask address as well. You can do this with a BNB or ETH faucet. To get BNB you have to visit the BSC faucet and send some BNB to your address. The same goes for ETH. You have to visit an ETH faucet for example the Rinkeby faucet and send some ETH to your address.

Create a BEP-20/ERC-20 token

Now you are completely set to create a BEP-20/ERC-20 token. First go to https://coinmanufactory.com/. A popup will come up which will ask you to connect to this site. Please click connect. If this popup doesn't come up you are either already connected to the website or it will come up when you create the token.

Now you can choose wether you like to create a BEP-20 or ERC-20 token and pick the token you would like to create. For this example we are creating a Basic BEP-20 token.

You have to enter the required details for you token and then click on create.

A popup will come up and ask you to confirm the transaction. Please click confirm.

It will take some seconds and then you have deployed your own crypto token.

Add token to your wallet

After the token was successfully created a popup will come up which suggests you to add the token to your wallet. Click "ADD TOKEN" and your token will be automatically added to your wallet.

If the popup does not come up you have to add the token manually. First copy the contract address.

Now click on "ADD TOKEN" in the MetaMask-Extension. Here you have to enter the token address you just copied and click "Next".

The token is now added to you wallet.

Congratulations you created your own crypto token!
